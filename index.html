Point-Query Quadtree for Crowd Counting, Localization, and More

ICCV 2023 Implementation

About the Project

This repository implements the Point-Query Quadtree model, also known as PET (Point Query Transformer). It offers an intuitive, flexible approach for crowd counting and localization by adaptively querying points in images.

For more details, view the paper on arXiv or download the full ICCV 2023 paper below.

Download the ICCV 2023 Paper (PDF)

Visit the GitHub repository: Milisav415/PET_crowd_counting

Highlights

  • Decomposable point querying for flexible crowd modeling.
  • Adaptability to both sparse and dense crowd scenes.
  • State-of-the-art performance on crowd counting and localization tasks.
